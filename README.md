# pizzaAPI
A RESTful API for Pizza Web Application.

# How to Run
Install Node first.  Then clone this repo.  After that either run:
`npm start `

or 

`nodemon`.

NPM packages should be preinstalled.  If not, please do:

`npm install` 

before initial run.
